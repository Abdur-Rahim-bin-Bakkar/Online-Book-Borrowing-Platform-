# 📚 Book Library Web App

## 🌐 Live URL
https://book-library-two-kohl.vercel.app/

---

## 🎯 Project Purpose

The **Book Library Web App** is a modern web application designed to manage and browse books efficiently.  
Users can explore books, search and filter them, and borrow books using an authentication system.

This project is built as a learning and practice project for **Next.js, authentication, and full-stack development concepts**.

---

## ✨ Key Features

- 📖 Browse all available books
- 🔍 Search books by title
- 🗂️ Filter books by category
- 👤 User authentication (login & signup)
- 📦 Borrow book functionality
- 🔒 Protected actions (only logged-in users can borrow)
- ⚡ Fast and responsive UI
- 🔔 Toast notifications for user feedback
- 🔄 Session-based authentication handling

---

## 🛠️ Tech Stack

- Tailwind CSS
